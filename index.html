Operators in JavaScript

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations.

Operators:

+, -, *, /, %



Assignment Operators

Assignment Operators are used to assign values of variables.

Operators:

==, +=, -=, *=, /=, %=, **=, &=, |=, ^=, <<=, >>=, >>>


Relational Operators

Relational Operators used to compare values.

Operators:

==, ===, !=, !==, >, >=, <, <=



Logical Operators

logical operators are used to combine multiple conditions or values and return a boolean result (true or false).

Operators:

&&, ||, !
